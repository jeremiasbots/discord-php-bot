# Discord Bot in PHP

This is a simple example of a Discord bot written in PHP.

## Setup

1. **Create a `.env` file** in the project root and define your bot token:
   ```env
   BOT_TOKEN=your_bot_token_here
   ```
2. **Start the bot** using Docker Compose:
   ```sh
   docker compose up
   ```
